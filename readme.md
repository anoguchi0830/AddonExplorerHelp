Addon Explorer は、日本語または韓国語のキーワードを使って Adobe Express 向けのアドオンを見つけやすくするための軽量な検索ツールです。

マーケットプレイスに掲載されている多くのアドオンは主に英語で説明されているため、日本語や韓国語で検索するユーザーにとっては、関連するツールを見つけることが難しい場合があります。

Addon Explorer は、ローカライズされたメタデータを含むアドオンカタログを提供し、アドオンを検索しやすくします。ユーザーは母国語で検索したり、カテゴリ別にアドオンを閲覧したりして、マーケットプレイスで該当するアドオンを簡単に見つけることができます。

このアドオンは、シンプルなパネルUIを通じて、アドオンの発見をより簡単にすることを目的としています。

Help page: https://github.com/anoguchi0830/AddonExplorerHelp/blob/main/readme.md

---
Addon Explorer is a lightweight discovery tool that helps users find add-ons available for Adobe Express using Japanese or Korean search keywords.

Many add-ons in the marketplace are primarily described in English, which can make it difficult for users searching in Japanese or Korean to discover relevant tools.

Addon Explorer provides a searchable catalog of add-ons with localized metadata. Users can search using their native language, browse add-ons by category, and easily locate the corresponding add-on in the marketplace.

The add-on is designed to improve add-on discoverability while keeping the experience simple and fast within the panel interface.

Addon Explorer는 일본어 또는 한국어 키워드를 사용하여 Adobe Express에서 사용할 수 있는 애드온을 보다 쉽게 찾을 수 있도록 도와주는 가벼운 검색 도구입니다.

마켓플레이스에 등록된 많은 애드온이 주로 영어로 설명되어 있기 때문에 일본어 또는 한국어로 검색하는 사용자에게는 원하는 도구를 찾기 어려울 수 있습니다.

Addon Explorer는 로컬라이즈된 메타데이터가 포함된 애드온 카탈로그를 제공하여 애드온을 보다 쉽게 검색할 수 있도록 합니다. 사용자는 자신의 언어로 애드온을 검색하거나 카테고리별로 탐색하여 마켓플레이스에서 해당 애드온을 쉽게 찾을 수 있습니다.

이 애드온은 단순하고 빠른 패널 인터페이스를 통해 애드온 발견 경험을 개선하는 것을 목표로 합니다。
